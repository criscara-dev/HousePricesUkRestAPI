Documentation

Introduction

Welcome to the UK House Prices API;
This documentation should help you familiarise yourself with the resources available and how to consume them with HTTP requests. Read through the getting started section before you dive in. Most of your problems should be solved just by reading through it.

Getting Started

Let's make our first API request to the UK House Prices API. Open up a terminal and use curl or Postman to make an API request for a resource. In the example below, we're trying to get the first House in the collection, identified by the numeric Id:

"http://127.0.0.1:5000/house/1

Base URL

The Base URL is the root URL for all of the API, if you ever make a request and you get back a 404 NOT FOUND response then check the Base URL first.
The Base URL for the UK House Prices API is: 

"http://127.0.0.1:5000/

Authentication

UK House Prices API is a completely open API. No authentication is required to query and get data. This also means that we've limited what you can do to just GET-ing the data. If you find a mistake in the data, then tweet the author or email him.

Searching

Since the DB size and related features, all resources support pagination and a search parameter that filters the set of resources returned. This allows you to make queries like:

"http://127.0.0.1:5000/houses?page=3&size=10

All searches will use case-sensitive matches on the set of search fields.

JSON

JSON is the standard data format provided by the API by default.

Attributes

  • Idstring -- The numeric Id for the House data.
  • Code string -- The Code Identifier from the HM Land Registry.
  • Price string -- The price at which the house has been sold last time.
  • Date string -- The date the house has been sold the last time.
  • Postcode string -- The house Postcode.
  • Property Type string -- The house type property; possible values are:
    • D for: detached
    • S for: semi-detached
    • T for: terraced
    • F for: flat/maisonette
    • O for: other
  • New built? string -- The house type property; possible values are:
    • True for: new-build
    • False for: not new-build
  • Estate Type string -- The house Estate type property; possible values are:
    • F for: freehold
    • L for: leasehold
  • House/flat number string -- The house or flat number.
  • Street Address string -- The house street address.
  • Town string -- The Town in which the house is located.
  • District string -- The District in which the house is located.
  • County string -- The County in which the house is located.
