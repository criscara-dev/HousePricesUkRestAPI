What is this?

The UK House Prices API is the UK's first quantified and programmatically-accessible data source for all the data from the HM Land Registry.
We've taken all the rich contextual data from the HM Land Registry ina form of a csv file, and formatted into something easier to consume with software.
Then we build upon an API on the front so you can access it.

We've formatted this data in JSON and exposed it to you in a RESTish implementation that allows you to programmatically collect and measure the data.

Check out the documentation to get started consuming swapi data

What are the features?

We're using Flask REST Framework to serve a RESTish API to you. The data is all formatted in JSON
